# Qr-code-generator
This is a very basic QR Code Generator. 

Note: The website does not look pretty because I have designed it all by myself.

How is it built-
                Basic HTML and css using features like flexbox, select, option etc.
                Used basic javascript for writing the logic into downloading/copying to clipboard and selecting qr size.

How it works is- 

                1. Paste a text/URL in the box
                2. Select the size in which you want your QR code.
                3. Click on Generate QR Code
                4. Download/Copy to clipboard- do as you like.
